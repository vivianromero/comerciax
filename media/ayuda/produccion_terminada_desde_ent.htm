Producción terminada desde entidades externas

 Top  Previous  Next

Se realiza l arecepción de producción terminada desde entidades externas, o sea, de aquellos cascos que se trasladaron a otras recapadoras del país y fueron procesados en ellas.

 

 

Al salvar los datos se presenta la pantalla con los datos captados y la opción para adicionar los cascos que se entregarán a producción terminada desde entidades externas.

 

 

 

Al seleccionar la opción se presentará la pantalla para la selección de los cascos que pertenecen al documento captado.

 

 

Se presentan los cascos que están transferidos para la entidad externa seleccionada en el documento y que aún no han regresado.

 

Se selecciona la medida de salida para los cascos que se seleccionen, si no se selecciona una medida de salida está se asume como la de entrada.

 

Si se selecciona los datos son salvados y se continúa la captación.

 

Se oprime

 

 

Si se edita un casco se podrá cambiar la medida de salida y el numero externo, o sea, el número que se le dio en el lugra donde se procesó.
