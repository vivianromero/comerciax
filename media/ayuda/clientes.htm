Se gestionan los clientes y la información del contrato.

 

 

El campo Externo se marca para los clientes que son otras recapadoras del país.

Se introducen los datos del cliente, al salvar se presenta la pantalla que muestra los datos y la opción de adcionar el contrato.

 

 

Si se selecciona la opción se presenta la pantalla para captar los datos del contrato.

 

 

Comportamiento Para la Venta  significa que son los clientes que entregan cascos para que la recapadora los venda.

 

 

Una vez entrado los datos del contrato se selecciona Salvar, se salvan los datos y se regresa a la pantalla anterior.

 

 

Si se selecciona la opción se muestran los datos del contrato.

 

 

El cotrato se puede Eliminar, Cancelar, Editar

Se le adiconan los Representantes y los Transportadores.

 

 

Cuando un contrato está Cancelado se presenta sombreado por un color

 

 

El cliente puede tener un solo contrato activo, si trata de insertar otro el sistema envia el mensaje siguiente y no deja que sea adicionado otro contrato.
