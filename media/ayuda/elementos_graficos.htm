            Los siguientes botones se muestran en las pantallas de captación de datos

 

           

              Salvar los datos que se están captando, se realiza la validación de los mismos, si no hay error se salvan y se regresa a la pantalla anterior, en caso contrario se emite el mensaje de error y no se salva.

 

 

             

                  Salvar los datos que se están captando, se realiza la validación de los mismos, si no hay error se salvan y se continúa la captación de nuevos datos, en caso contrario se emite el mensaje de error y no se salva.

                   

             

              Se cancela la operación de captación de datos, no se salvan y se regresa a la pantalla anterior.
