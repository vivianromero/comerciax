Se realiza la factura al cliente.

 

 

El número de la factura es generado automáticamente.

 

El Tipo Ajuste, se presentan aquellos cascos que en Recepción a Clientes se entraron por este mismo tipo.

El Tipo De Venta, se presentan aquellos cascos que en Recepción a Clientes entraron por el tipo Para la Venta.

El Tipo Otro, se presentan aquellos cascos que en Recepción a Clientes entraron como Otros.

 

Al salvar los datos se presenta la pantalla con los datos captados y la opción para adicionar los cascos.

 

 

Una factura puede ser editada o eliminada siempre y cuando no se halla confirmado.

La factura podrá imprimirse solo cuando se halla confirmado. Cuando se confirma una factura no se puede eliminar, se cancela.

 

El Precio CUP y Precio CUC dependen de los establecido con el cliente en el contrato.

 

A una factura se le pueden añadir los cascos a partir de una oferta si se presiona , se presentan las ofertas del mismo tipo realizadas al cliente para seleccionar la deseada.

 

 

Si se presiona se mustran los datos de la oferta

 

 

Al seleccionar la opción se presentará la pantalla para la selección de los cascos que serán facturados. Los cascos que se presentan son aquellos que entraron por el mismo tipo que se realiza la oferta y se encuentran en producción terminada.
